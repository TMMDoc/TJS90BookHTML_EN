Installing the Software

Installing the Software

Pre-requisite

TJS pre-setup should be installed once before initiating the TJS installer. This will install .Net 8 and redistributables required by JFYBend and TJS.

Click the following link to download the pre-setup file:
TJS Pre-setup

License

A serial key is required to activate TJS License. This serial key is a 16 digit alpha numerical characters. See below for example:

Example

1A34567B : 876C4321

For new license, send a request mail from your official mailing address to balakrishnan.madurakavi@trumpf.com

Online activation

On opening TJS, a registration window appears.

Figure 1. Registration window

  • Make sure an active internet connection is required for activating the license.

  • To register, enter your username and serial key.

Offline activation

If the internet connection is not available or the license server is not reachable due to a firewall, offline activation is possible.

  • Enter the serial key in the registration window. Connection to server failed error message will be displayed as shown below:

  • The TJS.request file is found in folder C:\users\<username>\AppData\Local\TJS.

Local computer username should be used in <username> in the above folder path.

  • Send a copy of this file to balakrishnan.madurakavi@trumpf.com along with the serial key used.

  • In return you will receive a TJS.db file. Place it in the above folder location.

  • Try registering again with your username and serial key, and the TJS license will be activated.

Network License

A network license can be used to run TJS in a system (which is connected to an active network).

  • To use a network license, the following line should be added in middleware.ini located in C:\ProgramData\Metamation\RA\RData:

[Basic]

LicenseServer=<IP addresss>
This requires the installation of a license server (MetaLock).

Screen Size

  • To adjust the screen size, navigate to Settings → Other Settings → Screen Size.

  • Enter the desired screen dimension and click the tick mark.
