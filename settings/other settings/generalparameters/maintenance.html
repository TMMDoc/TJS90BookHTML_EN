Function

This parameter enables/disables the endurance mode for testing of the machine.

Description

When this parameter is enabled and the foot pedal close is pressed, the complete bending cycle runs continuously until the foot pedal is released. This mode is useful for testing the machine in floor line.

Additional in Settings Maintenance

When user is in maintenance page with access level-service the following options appears:

  1. Reset time: Used to reset hours, strokes and Pump on time

  2. Save backup: Used to save the machine backup to selected location

  3. Load backup: Used to load the selected machine backup file

  4. Load other backup: Used to load backup file from a different machine

  5. Load PLC: Loads the PLC program (bending.app file)

  6. License: Used to view license information - Serial key, Name Licensed to, features included.
