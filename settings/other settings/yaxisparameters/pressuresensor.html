Pressure Sensor

Pressure balance

Pressure balance enable

Range

None/Electronic/Hydraulic

Default

Electronic

Units

Access

Administrator

Function

This parameter activates/deactivates the pressure balance function on the Y-axis.

Description

With the Pressure Balance feature, the pressure difference over the proportional valves is kept constant. For an electronic pressure balance three pressure sensors need to be mounted on the machine. One to measure the system pressure and one for each Y-cylinder. If this feature is enabled, the other parameters on this page need to be programmed. They are described in the following section.

Feedback

Pressure at full scale of sensor

Range

1-999

Default

400

Units

Bar

Access

Administrator

Function

This parameter defines the maximum pressure value that can be measured from the pressure sensor.

Description

The pressure sensor feedback to the controller corresponds to 0 - 10V max. This value refers to maximum pressure at 10V signal. This parameter is valid for all the three sensors used for the electronics pressure balance function.

Control

Balance pressure

Range

1-999

Default

28

Units

Bar

Access

Administrator

Function

This parameter defines the pressure value that will be added to the maximum of the measured pressure in the upper side of the cylinders.

Description

NA

Control margin

Range

1-100

Default

25

Units

Percentage

Access

Administrator

Function

This parameter defines the allowed control range for the pressure control loop.

Description

The maximum deviation from the original set point is defined by this parameter. If this value is too small, the desired pressure will not be reached. The value that should be programmed also depends on how well the pressure valve has been calibrated. If a value larger than 40% needs to be programmed, the pressure valve calibration should be checked.

P-gain

Range

0.001 - 1

Default

0.2

Units

NA

Access

Administrator

Function

This parameter defines the proportional gain for the pressure valve control for faster and accurate response.

Description

With the P-gain the pressure valve control can be optimized. A larger value will give a more accurate and faster response. When set too high, oscillation can occur.
