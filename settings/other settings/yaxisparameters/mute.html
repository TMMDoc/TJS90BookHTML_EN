Mute

Mute

General

Mute programmable each step

Range

ON/OFF

Default

ON

Units

Access

Administrator

Function

This parameter enables/disables the editing of mute parameter for each bend in the program edit cycle.

Description

When this parameter is ON, the mute point can be changed for each individual bend of a program.

Mute changed tolerance

Range

0 - 9.99

Default

0.01

Units

mm

Access

Administrator

Function

This parameter defines a tolerance for the 'Mute changed' signal in the PLC.

Description

If there is a difference in the programmed and actual (from Laser safe) Mute point, a PLC flag is set within the RA software to signal the Mute has changed. If this signal is not necessary for small changes of Mute, a tolerance can be programmed with this parameter. If the difference between new and old Mute point is larger than this tolerance, the flag is set. This parameter is irrelevant if the flag is not used in PLC.

External mute adjust

Range

0-99

Default

0

Units

mm

Access

Administrator

Function

This parameter defines the mute distance additional to computed value.

Description

To program an alternative mute position intended to stop the beam at a higher mute position.

Minimum relative mute

Range

0-99

Default

6

Units

mm

Access

Administrator

Function

This parameter defines the minimum mute value of every V-die.

Description

The mute value that is programmed in the parameters of every V-die, is limited to this machine parameter value. The operator cannot program a value that is lesser than this parameter.
