Machine Settings

This document explains the different machine parameters used to control a press brake.

Changing User Levels

From Settings page navigate to Other settings → Machine settings → Change access.

The most basic user level is Standard, which allows you to change the fundamental machine parameters. In this level, you cannot change the machine settings denoted by or .

To change the machine parameters denoted by , you must be a administrator or service user.

Service level user access denoted by enables for changes to all parameters. Normally, these parameters are set to the factory defaults and requires no change.

To change the user level access select the correct user level and enter the password.
